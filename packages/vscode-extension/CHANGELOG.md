# Changelog

## 1.0.0 (2026-03-28)

- Initial release
- PDF + DOCX export from Markdown
- 5 themes: Clean, Dark, Editorial, Technical, Warm
- 5 font families: Sans, Serif, Mono, Humanist, Slab
- Right-click context menu export
- Export settings panel with live preview
- Cover page generation with optional logo
- Table of contents with anchor links
- Configurable margins, line spacing, headers, footers, page numbers
- License key management via VS Code SecretStorage
